# pocket video synthesizer — operator's guide

A user manual for [po10diy](https://github.com/ramonaisonline/po10diy), Ramona Sharples' pocket-operator-format video synthesizer.

**Read it:** https://0xbigdan.github.io/po10diy-guide/

One self-contained HTML file, written from the firmware source. Covers the stock firmware: panel layout with an annotated diagram, power and quick start, the 8 layers with knob and variant mappings, sequencing, the 8 fx, the on-screen UI, outputs, known quirks, specs, and what's still on the to-do list.

Hardware, PCB files, firmware, and build instructions live in the upstream repo — this is just the manual.
